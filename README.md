# E2HTranslatorMac

E2HTranslatorMac은 영어 단어를 포함한 문장을 입력받아, 영문 앱 이름이나 고유 명사는 그대로 유지하면서 나머지 문맥을 자연스러운 한국어로 변환해 주는 macOS 전용 번역 애플리케이션입니다.

## 설치 방법 (Installation)

1. [Releases](https://github.com/mrahn80/E2HTranslatorMac/releases) 페이지에서 최신 `.dmg` 파일을 다운로드합니다.
2. 다운로드한 `E2HTranslatorMac.dmg` 파일을 엽니다.
3. 앱 아이콘을 `Applications(응용 프로그램)` 폴더로 드래그하여 설치합니다.
4. Launchpad 또는 응용 프로그램 폴더에서 앱을 실행합니다.

## 개발 환경 (Development)

* macOS 13.0+
* Xcode & Swift 5.0
* XcodeGen을 활용한 프로젝트 관리 (`project.yml`)
